# sanskrit-guna-ai-alignment

Cross‑cultural AI safety mini‑project using Samkhya (Sāṅkhya) philosophy. Short research sprint (≈10 days of focused work) to build a small Sanskrit guna dataset (sattva/rajas/tamas), train a simple classifier, benchmark LLMs, and expose a Streamlit demo as a cross‑cultural alignment component.

## Goals (Minimum Credible Research)

- Create a small but real Sanskrit guna dataset (≈200–500 labeled examples).
- Train a simple baseline classifier (e.g., scikit‑learn) for Sattva/Rajas/Tamas.
- Evaluate and compare one or more general‑purpose LLMs on the same task.
- Build a minimal Streamlit app that takes text input and returns guna + confidence + alignment decision.
- Draft a short 4–6 page arXiv‑style report describing problem, data, methods, results, and future work.
